# Readme File

This is a sample readme file for the assignment. It contains basic information about the project.

## Instructions

1. Create a directory named `lib`.
2. Inside the `lib` directory, create three files: `readme.txt`, `students.csv`, and `index.html`.
3. Populate these files with the provided content.
4. Write a function called `getFileContent(file)` to read and return the content of the specified file.

## Files

- `readme.txt`: This file.
- `students.csv`: Contains student data.
- `index.html`: Basic HTML page.

## Usage

Use the `getFileContent` function to read the content of each file and display it in the console.
